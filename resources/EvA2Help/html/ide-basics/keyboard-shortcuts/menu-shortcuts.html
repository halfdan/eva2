Menu Shortcuts

There are keyboard shortcuts available to activate the following menu commands and other general commands:

  • Options menu

Options menu

Keys Command Action
CTRL+P Preferences Opens the preferences dialog
CTRL+Q Quit Exits the application

Windows menu

Keys Command Action
CTRL+M Cascade Windows Cascades the visible internal windows
CTRL+, Tile vertically Tile the visible internal windows vertically
CTRL+. Tile horizontally Tile the visible internal windows horizontally

Help menu

Keys Command Action
F1 Help Opens the help contents
